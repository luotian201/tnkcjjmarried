一个专为陶能科和陈金金定制的喜庆结婚邀请网站。

## ✨ 特性

- 📱 **移动端优化** - 完美适配手机浏览
- 🎨 **喜庆设计** - 红色主题，渐变背景
- 🖼️ **自动轮播** - 支持多张照片自动播放
- 💕 **个人信息** - 新人姓名、结婚日期、地点
- 🚀 **一键部署** - GitHub + Vercel 免费托管

## 🚀 快速开始

### 1. 本地预览
```bash
npm install
npm run dev
```
访问 http://localhost:3000

### 2. 添加照片
将照片放入 `public/photos/` 文件夹，支持 jpg/png/webp 等格式。

### 3. 部署到网上
1. 推送到 GitHub
2. 连接 Vercel 一键部署
3. 获得免费域名分享
